# C Fundamentals

This repository contains a set of C programming exercises completed during the early stage of learning programming.

## Topics Covered
- Arrays and Sorting
- System Calls (Process Creation)
- File Handling
- Dynamic Memory Allocation

## Projects Included
1. Array Sorting in Descending Order
2. Process Creation using fork()
3. Directory Listing using opendir() and readdir()
4. Memory Allocation Simulation (MVT)

## Concepts Learned
- Basics of C programming
- Operating System concepts
- File system interaction
- Memory management

## Purpose
This repository represents my foundational stage in C programming before moving into advanced software development and system-level projects.
