# Memory Allocation Simulation

## Description
This program simulates contiguous memory allocation by accepting the total available memory and the memory requirements of multiple processes. It displays the allocated memory and the remaining external fragmentation.

## Concepts
- Dynamic memory allocation concepts
- Memory management
- External fragmentation
- User input

## Input
- Total available memory
- Memory required for each process

## Output
- Memory allocated to each process
- Total allocated memory
- Remaining external fragmentation
